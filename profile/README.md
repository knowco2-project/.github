# knowco2 — Know your air. Know your focus.

**knowco2** is an open-source hardware + software project for monitoring indoor CO₂ and helping people make better ventilation decisions.

> CO₂ is a *proxy* for ventilation effectiveness. When CO₂ rises, it often means rebreathed air is building up.

## What we’re building
- **Portable CO₂ monitor** (device + firmware)
- **Local-first UX** (device AP mode for setup + dashboards)
- **Optional cloud** (secure ingestion + history + alerts)
- **Docs that make it easy** (build, flash, calibrate, deploy)

## Repositories (start here)
- **Firmware** → ESP32-S3 + SCD4x (CO₂ / temp / humidity)
- **Hardware** → case, BOM, wiring, mechanical files
- **Cloud** → ingestion, auth, storage, cost controls
- **Apps** → iOS / web dashboards

## Project principles
- **Accurate sensors, honest UX**
- **Private-by-default**
- **Simple to build**
- **Low operating cost**
